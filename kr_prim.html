Известные исторические шифры.

Шифр Атбаш (древнееврейский)

Один из древнейших известных шифров. В нём алфавит просто переворачивается: первая буква становится последней, вторая — предпоследней, и так далее.

Принцип: A ↔ Z, B ↔ Y, C ↔ X, ...

Слово "HELLO" → "SVOOL"

Шифр Цезаря

Простой шифр замены, при котором каждая буква сдвигается на определённое число позиций в алфавите. Назван в честь Юлия Цезаря, использовавшего его для военных сообщений.

Сдвиг на 3: A → D, B → E, C → F ...

Слово "ABC" → "DEF"

Символика майя

Майя использовали иероглифическую письменность, где каждый символ мог означать слог, слово или даже фразу. Некоторые символы использовались тайно для записи священных или правительственных текстов.

Их письменность можно считать криптографичной — для чтения требовалось знание культуры и кода символов.

Арабская криптография

Учёные арабского мира в Средние века не только использовали шифры, но и изучали их математически. Один из первых трактатов по криптоанализу был написан Аль-Кинди в IX веке.

Аль-Кинди применял частотный анализ — сравнение частоты появления букв для взлома шифров.

Шифровальная машина "Энигма"

Использовалась Германией во Второй мировой войне. Это была механическая машина, которая постоянно изменяла соответствие между буквами по сложной системе вращающихся дисков.

Энигму удалось взломать благодаря усилиям криптоаналитиков, включая Алана Тьюринга. Это стало важным шагом в развитии современной криптографии.
